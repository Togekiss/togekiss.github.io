**Infoboxes** are a staple of all wikis: Tables that float on the right of the page and show a basic summary of the information of the article.

The infobox on the right is a real element! Inspect it or the Github code to see it.

Since my users are young writers, not developers, I have built these templates making them as easy to use as possible without sacrificing features. To achieve this, the extension **TemplateData** allows me to build an interactive form that editors can fill when using the View Editor; and renders a detailed table listing the fields in the documentation. With this I can fill the form with instructions, examples, and even predefined sets of options for editors to choose with a dropdown.
